Bootstrap script for {{ agent_name }}

Run this on the target box. Token expires in 1 hour.

mkdir -p /etc/gofim {{ scan_path }}/.gofim
cat > /etc/gofim/gofim.yml <<'EOF'
agent_name: {{ agent_name }}
path: {{ scan_path }}
{%- if excludes %}
exclude:
{%- for e in excludes %}
  - '{{ e }}'
{%- endfor %}
{%- endif %}
server_url: {{ server_url }}
EOF

go-fim -c /etc/gofim/gofim.yml --setup {{ token }}

# Then schedule the scan loop, e.g. with cron:
#   */5 * * * *  go-fim -c /etc/gofim/gofim.yml

Re-running the --setup line is a no-op once the agent is registered, so it's safe to paste twice. After setup succeeds, the agent will appear on the dashboard.
